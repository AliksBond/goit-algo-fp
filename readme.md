# Завдання 7. Використання методу Монте-Карло

## Опис завдання

Програма імітує кидки двох кубиків і обчислює ймовірність кожної можливої суми (2–12) методом Монте-Карло.  
Результати порівнюються з аналітичними ймовірностями.

- Сині стовпчики — ймовірності, отримані методом Монте-Карло.  
- Червона лінія з точками — аналітичні ймовірності.  
- Графік показує, що результати Монте-Карло майже точно відтворюють теоретичний розподіл.

## Висновки

- Метод Монте-Карло ефективно моделює випадкові процеси.  
- Отримані ймовірності сум кубиків дуже близькі до аналітичних.  
- Зі збільшенням кількості симуляцій точність наближення підвищується, що підтверджує коректність реалізації.
